# jquery-xImage


###在页面引入jquery.xImage.js

###调用方式根据需求来选择，一共提供了如下几种：

type 1：
`$('#ImageBox0 img').xImage({  //style 0 
		animated : 'fade'  
	});`
  
 type 2:
 `$('#ImageBox1 img').xImage({  //style 1 
		ImageNav : '#ImageNav1 li',  
		ImageTit : true,  //打开图片标题显示
		ImageBoxParent : '#ImageCon1',  //父级
		animated : 'none'  
	}); `
